# C++面向对象

使用cmake构建侯捷老师上课的样例。目前完成了第一课的所有例子。course是所有样例。course/test是样例的测试文件

## 编译文件

下载工程，执行如下命令
```C++
./build.sh
```
在当前工程目录的build/release/lib文件下为course的所有类包，
build/release/bin为测试的可执行文件，如complex_test,在不同操作系统中生成的不一样。可以直接运行
